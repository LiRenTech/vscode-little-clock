# little-clock

A simple and intuitive timer extension for Visual Studio Code.

## Features

- **Dynamic Timer Display**: Shows only relevant time units (seconds, minutes, hours)
- **Status Icons**: Visual feedback with different icons for initial, running, and paused states
- **Click Interaction**: Easy toggle between start, stop, and reset states
- **Clean UI**: Minimalistic design that integrates seamlessly with VS Code

## Usage

1. The timer appears in the bottom right corner of the VS Code status bar
2. Click the timer to start it
3. Click again to stop it
4. Click again to reset it
5. Click again to start a new timer

## Timer States

- **Initial State**: Shows `$(clock) 00` with a clock icon
- **Running State**: Shows `$(play) MM:SS` or `$(play) HH:MM:SS` with a play icon
- **Paused State**: Shows `$(debug-pause) MM:SS` or `$(debug-pause) HH:MM:SS` with a pause icon

## Requirements

- Visual Studio Code 1.107.0 or higher

## Extension Settings

This extension currently has no configurable settings.
